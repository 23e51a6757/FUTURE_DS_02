# FUTURE_DS_02 – Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer churn using the Telco Customer Churn dataset and presents key insights through an interactive Power BI dashboard.

## Objective
- Analyze customer churn patterns.
- Identify key factors affecting customer retention.
- Build an interactive dashboard for business insights.

## Tools Used
- Power BI
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
**WA_Fn-UseC_-Telco-Customer-Churn.csv**

## Dashboard Features
- Total Customers
- Churned Customers
- Retained Customers
- Churn Rate
- Contract Type Analysis
- Internet Service Analysis
- Payment Method Analysis
- Gender-wise Distribution
- Interactive Slicers

## Key Insights
- Month-to-Month contract customers have the highest churn.
- Fiber Optic users show higher churn.
- Electronic Check customers have a higher churn rate.
- Long-term contracts improve customer retention.
